# 福利彩票平台使用说明书

本文档旨在指导管理员、店主及用户如何正确使用本平台。

---

## 📖 业务操作流程

### 店铺入驻及使用流程

** 💡 注意：请务必按照以下流程顺序操作，以确保各项功能展示正常 **

#### 1. 管理员：后台录入店铺信息
- **操作端**：管理后台 `https://we.lansess.com`
- **操作步骤**：
    1. 进入“店铺” -> “店铺列表” -> “创建店铺”。
    2. 填写店铺信息（**手机号必须填写正确**，用于后续店主登录），初始状态为“审核中”。
    3. 勾选列表数据前面的复选框，点击“审核通过”按钮，状态转为“正常”。
    4. 店铺通过审核后，店主端方可登录。
    5. 将生成的“邀请码”提供给店主。

#### 2. 店主：注册与彩种配置
- **操作端**：店主端 App
- **操作步骤**：
    1. 下载店主端 App 并登录（需使用管理员录入的手机号）。 登录方式为手机号登录，手机号用正常手机号登录（测试阶段验证码用点击“获取验证码”后提示的，不需要真实短信验证码）。
    2. **重要信息**：您的专属“邀请码”会直接在 App 顶部或设置页面中展示。
    3. 登录后，进入“设置”页面配置支持的“彩种”，勾选即可生效。
    4. 请妥善保存并管理您的邀请码，后续需提供给进店购彩的用户。

#### 3. 用户：登录与店铺绑定
- **操作端**：用户端 App
- **操作步骤**：
    1. 下载用户端 App 并登录。登录方式为手机号登录，手机号用正常手机号登录（测试阶段验证码用点击“获取验证码”后提示的，不需要真实短信验证码）。
    2. 进入“店铺”页面，输入店主提供的“邀请码”进行绑定。
    3. 绑定成功后，返回首页。当首页能正常展示彩种列表时，说明设置成功。

### 🛍 交易操作流程

#### 用户下单流程
1. **选择彩种**：在首页选择感兴趣的彩种（如：双色球、大乐透等）。
2. **选号确认**：进入选号页面，按照规则选择号码或使用随机选号。
3. **提交并支付**：确认订单金额及选号内容，完成在线支付。
4. **等待出票**：支付成功后，订单会自动发送至绑定的实体店中。

#### 店主接单出票流程
1. **收到新单**：店主端 App 收到新订单提醒（状态为“已付款”）。
2. **确认接单**：店主点击“接单”，表示准备出票。
3. **机具出票**：店主在店内实体彩票机上完成手动出票。
4. **回传票样**（**关键环节**）：店主点击“拍照出票”，拍摄清晰的实体草票照片并上传。
5. **交易完成**：上传成功后，订单状态变为“已出票”，用户即可在 App 中查看到真实的票样。
